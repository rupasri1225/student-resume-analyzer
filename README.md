📄 Student Resume Analyzer

🚀 Description

Student Resume Analyzer is an AI-based project that analyzes a student's resume and provides useful insights such as skill evaluation, suggestions for improvement, and overall resume feedback. It helps students enhance their resumes to match industry standards.

---

🎯 Features

- 📌 Extracts text from resume
- 🤖 Analyzes skills and keywords
- 📊 Provides improvement suggestions
- 🧠 AI-based evaluation
- ⚡ Simple and user-friendly interface

---

🛠️ Tech Stack

- Python
- Machine Learning / NLP
- Streamlit (if used) / Flask (if used)
- HTML, CSS

▶️ How to Run

1. Clone the repository

git clone https://github.com/rupasri1225/student-resume-analyzer.git

2. Navigate to project folder

cd student-resume-analyzer

3. Install dependencies

pip install -r requirements.txt

4. Run the app

python app.py
